# Berbagai Masalah

## **Mengklik Unduh Tidak Mengunduh Arkaik**

**Masalah:**

* Baik menggunakan installer atau file ZIP, **Arkaik sama sekali tidak diunduh**.

**Solusi:**

* Cari **"Pengaturan Tablet PC"**
* Di opsi pertama, atur ke **"Jangan pernah gunakan mode tablet"**
